Current Research

Ongoing · HCI Lab

Privacy UX &
Interface Control

Two parallel studies examining how cookie consent design shapes user behavior, and how UI affordances generate perceived — but not actual — agency.

Methodology

User experiments · Behavioral logging · Comparative interface testing · Regulatory analysis

Study 01 · 2024–2025

Cookie Consent as Performance: Autonomy, Dark Patterns, and the Limits of Notice

Empirical study examining how consent interfaces direct behavior toward data collection while presenting the appearance of user control. Regulatory framework analysis + user testing (40+ participants).

Study 02 · 2025

The Illusion of Control: How UI Affordances Construct False Agency

Investigates mechanisms by which interface design creates perceived control that does not correspond to actual system influence. Focus on algorithmic feeds and AI-mediated environments.
